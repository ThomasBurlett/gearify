# Gearify

React + TypeScript + Vite foundation for a weather-aware sports outfit planner.

## Stack

- React 19 + Vite 7 + TypeScript
- Tailwind CSS + shadcn/ui
- React Router
- Vitest + Testing Library
- ESLint + Prettier

## Getting started

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run lint:fix
npm run format
npm run format:write
npm run test
npm run test:run
```
